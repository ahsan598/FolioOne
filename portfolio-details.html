Project Overview

This project is a scalable and secure multi-tier web application deployed on AWS using Terraform. It features a custom VPC with isolated subnets, multiple application servers for load distribution, and a managed database layer. Security is enforced via security groups and IAM roles, while Terraform ensures automated, repeatable deployments.

Approach

Infrastructure as Code

Automated provisioning with Terraform to ensure repeatability and version control.

Custom VPC Design

Created public and private subnets for proper isolation of application layers.

Secure Networking

Configured security groups and IAM roles with least-privilege policies.

Key Features

Scalable Architecture

Multi-tier design with multiple application servers for load distribution and high availability.

Secure Networking

Custom VPC, subnets, and security groups to ensure network isolation and protect resources.

Role-Based Access Control

IAM roles and policies to enforce least-privilege access for users and services.

Managed Database Layer

Reliable RDS database with automated backups and monitoring for data integrity.

Automated Deployment

Fully reproducible infrastructure using Terraform scripts to minimize manual errors.
